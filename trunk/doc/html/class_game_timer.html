int GameTimer::getTimeElapsedInMilliseconds (  )

Returns the elapsed time in mulliseconds.

While the timer is running, this function will not return the same value in subsequent calls. If the timer is not running, this will return the time between the last call to start() and the last call to stop()

std::string GameTimer::getTimeElapsedString (  )

Returns a string of the time elapsed.

While the timer is running, this function will not return the same value in subsequent calls. If the timer is not running, this will return the time between the last call to start() and the last call to stop().

The string format returned is "[hours:]minutes:seconds.hundredths" Note that an 'hours' value of 0 will not be included in the string
